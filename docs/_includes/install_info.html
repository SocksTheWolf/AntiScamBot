It is really easy to setup ScamGuard and enable the protection it provides for your community. Just follow these simple steps!

  1. Join the TAG server via this link.
  2. Invite the ScamGuard bot using one of the methods below.
    • Selecting ScamGuard account in the Discord and clicking "Add To Server":
    • Install via this Discord app link.
    RECOMMENDED: Place the newly created ScamGuard role one level below your server's moderator role OR one level above your regular members role. This helps prevent Discord permission mismatches that may occasionally occur.
  3. Activate ScamGuard by typing /activate in the TAG Server!
